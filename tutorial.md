## Cara Jalanin Simulator Dealer
`requirements`
- PHP 7.5 + (pasti ada klo punya xampp)

### step 1
- buat folder baru di folder `htdocs` xampp
- di terminal folder baru ketik `git clone https://github.com/zakyislm/uas-basdat-smt2.git`
### step 2
- nyalain xampp
- ke phpmyadmin buat database baru
- import sql terus pilih file `schema.sql`
### step 3
- abis database jadi, buka vs code dan pilih folder baru yang dibuat di htdocs tadi
- buka terminal dan ketik `php -S localhost:1234`
